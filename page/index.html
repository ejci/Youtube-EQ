Instalation

  • Download extension (.zip)
  • Unzip downloaded file (.crx)
  • Type chrome://extensions in your Google Chrome browser
  • Drag and drop unziped (.crx) file to Google Chrome extensions tab

Road map

  • Auto volume
  • EQ presets (rock, classic, speech,...)
  • Save EQ settings per video or globally
  • More visual effects
  • Performance optimalization
  • Remove jQuery dependency

Known Issues

  • Resizing of video player doesn't work properly
  • Changing video quality doesn't work properly
  • Some videos don't have HTML5 stream so the EQ doesn't show up ( it's a feature not a bug :) )
  • Performance issues
